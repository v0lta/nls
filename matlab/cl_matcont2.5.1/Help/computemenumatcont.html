The compute menu in the Main window allows you to begin the computation of curves.

Command

What it does

Forward

starts the computation of a new curve in the forward* direction

Backward

 starts the computation of a new curve in the backward* direction

Extend

 extends the computation of a previous curve in the same direction

You can pause or stop the computation at any moment. It also possible to suspend computation by setting an appropriate pause mode.

*There are two directions in which a curve can be computed starting from a given initial point. Arbitrairly one of them is called 'forward direction' and the other one is called 'backward direction'.
