The dynamical systems list dialog box list all dynamical systems in your archives and allows you to load or manipulate them. You call upon the dynamical systems list dialog box with the Select|Systems|Edit/Load command in the Main window. The commands listed in this section operate on the dynamical system whose name is highlighted in the list (not always the current system). To highlight a specific system just click on its name.

 

Command

What it does

system|delete

deletes the highlighted system.

 

Button

What it does

load

makes the highlighted system the current system. This command closes the dialog box and loads the selected system.
edit displays the System specification dialog box for the highlighted system and allows you to change its specification. After you close the dialog box, the system you have edited becomes the current one.
