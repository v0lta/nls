The curves list dialog box list all previously computed curves for the current system and allows you to manipulate them. You display the curves list dialog box with the select|curve command in the Main window. The commands listed below operate on the curve whose name is highlighted in the list (not only the current curve). To highlight a specific curve just click on its name.

 

Command

What it does
curve|Rename asks for the new name of the highlighted curve and then renames it. The current curve remains the same. New name may be any text of any length.

curve|Delete

deletes the highlighted curve.

There are following buttons in the dialog box:

Button

What it does

Select

makes the highlighted curve the current one. In particular, types of initial point and curve are taken from the highlighted curve. This command closes the dialog box.
Cancel This command closes the dialog box leaving the current curve unchanged.
