The main window contains several menus to control MATCONT. It displays useful information about objects you are currently operating on

Menu commands

Select menu commands are used to specify and select objects.

Compute menu commands begin computation of curves.

Window menu commands allow you to open extra windows.

Type menu commands are used to set the type of a curve to be computed or its initial point.

Options menu commands allow you to set various options of processing and storing your data.

Help menu commands provide you with access to these Help-files.

 

Information fields

Class displays the type of dynamical system you are working with (in this version this is always ODE)

System displays the name of the current dynamical system.

Curve displays the name of the current curve.

Point type displays the type of the initial point on the current curve.

Curve type displays the type of the current curve.

Derivatives indicates methods used for computation of derivatives of the right-hand side.

Status displays the status of MATCONT (Ready, busy, ...).

Duration displays the duration of the last action.
