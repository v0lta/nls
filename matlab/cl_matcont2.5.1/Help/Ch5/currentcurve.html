A current curve is a curve from the current system. MATCONT always takes the most recently computed curve as the current curve (you compute curves by Compute|forward and Compute|backward commands in the Main window). You may change the current curve with the curves list dialog box opened by the Select|curve command in the Main window. The name of the current curve is displayed in the Curve information field in the Main window.

The following operations can be performed on the current curve only:

 

Redraw

is initiated by the Redraw command from the Graphic windows.
Change attributes is initiated by the Attributes/Variables on axes command from the Graphic windows.

MATCONT remembers the current curve and selects it each time you select a dynamical system or when you start MATCONT (which implies selection of the same dynamical system as it was at the moment of the last termination).
