A starter which produces necessary data to generate a Homoclinic-to-Hyperbolic-Saddle curve from a Bogdanov-Takens point (BT) has the following parameters that can be modified via the starter window.

Initial point

 

p

a list of parameter names and their values.
Parameter for the first step
  
Amplitude contains the amplitude of the initial homoclinic orbit.
Homoclinic parameters
  
T contains the value of half the stored time interval of the orbit.
eps0 contains the value the distance of the first orbit point to the equilibrium.
eps1 contains the value the distance of the last orbit point to the equilibrium.
Jacobian data
  
Increment Increment to approximate partial derivatives by finite differences
Discretization data
ntst the number of test intervals.
ncol the number of collocation points.
Monitor userfunctions
  
function names Monitor toggle {yes/no}
Monitor singularities
  
Singularity types Monitor toggle {yes/no}
