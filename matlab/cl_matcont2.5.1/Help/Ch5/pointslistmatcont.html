The initial points dialog box list all curves along with all special points detected on these curves and allows you to select any special point as an initial point for the next curve to be computed. You display the initial points dialog box by clicking on Select|Initial point in the Main window. The highlighted curve upon calling this window is the most recently computed one.

 

There are following buttons in the dialog box

Button

What it does

Select

Selects the highlighted point as an initial point for the next curve to be computed. This includes loading the appropriate starter and generator and opening their windows to allow you to set or change their parameters. To highlight a particular point just click on it. This command closes the dialog box.
Cancel This command closes the dialog box without selecting a new point..

If you double-click a point of the current curve and a result window is open, it will be indicated on the plot of the curve.
