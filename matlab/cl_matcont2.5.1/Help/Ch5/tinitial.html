The term point type refers to certain property of the point. For example, a point x is said to be of type equilibrium (or shortly, equilibrium point) for a dynamical system x'=F(x), if its coordinates satisfy the equation F(x)=0. The type of a point is checked by the appropriate starter or generator which computes the point. You set a type of an initial point of a curve by the Type menu commands in the Main window.
