The options menu in the Main window allows you to set the pause mode option.

Command

What it does

Pause

displays the pause mode dialog box where you specify the pause mode which controls at what points computation will be suspended.

Plot

displays the output mode dialog box where you specify after which number of computed points you want output to be displayed in the output windows every time.

Archive

displays the archive mode dialog box where you specify the number of untitled curves that are stored of each type. An untitled curve is a curve whose name was not changed by the user. When the number of untitled curves exceeds the limit, MATCONT automatically deletes the 'oldest' untitled curve to comply with the restriction, and overwrites it with the new computation results.
