A starter which produces the necessary data to generate the limitcycle curve from a hopf point (H, ZH, DH, GH) has the following parameters that can be modified via the starter window.

 

Initial point

 

x

a list of phase coordinate names and their values at the initial point.

p

a list of parameter names and their values.
period the period of the initial cycle.
Jacobian data
  
Increment Increment to approximate partial derivatives by finite differences
Switch data
  
Amplitude contains the amplitude of the initial cycle.
Discretization data
ntst the number of test intervals.
ncol the number of collocation points.
Monitor userfunctions
  
function names Monitor toggle {yes/no}
Monitor singularities
  
Singularity types Monitor toggle {yes/no}
Calculate multipliers
  
Multipliers
 Compute toggle {yes/no}
Calculate Phase Response
  
Input the strength of the input to which the PRC and dPRC are computed.
PRC
 Compute toggle {yes/no}
dPRC Compute toggle {yes/no}
