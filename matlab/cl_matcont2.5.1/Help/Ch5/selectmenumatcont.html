The select menu in the Main window allows you to choose various objects for processing, and to specify new objects and manipulate them.

Command

What it does

Systems

 gives you two options:
  • New: define a new system, supplying equations, parameters and phase variables
  • Edit/load: displays the system list dialog box with a list of dynamical systems stored in archives and allows you to manipulate them (select a system, edit a system, etc.)
Diagram allows you to select a create/change/switch the directory from which the archives of computed curves are taken.
Curve displays the curves list dialog box with a list of stored curves, which also allows you to select, rename or delete curves.
Initial point displays the intial points dialog box with a list of special points detected on curves, which allows you to take one of them as the initial point for a curve being computed next
User functions displays the user functions dialog box, which allows you to specify user functions whose values may be monitored along curves.
Exit closes MatCont, after asking for confirmation

MatCont remembers the current state of the dynamical system and curve after closing a session and selects them automatically after startup.
