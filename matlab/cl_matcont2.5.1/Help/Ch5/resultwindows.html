Plot windows are used to represent computed curves in various graphical forms. You open plot windows with the command Window|plot in the Main window.

There is the minimal set of plot windows:

Command

What it does

2D-plot

a window of this form is used to represent two-dimensional projections of curves.
3D-plot a window of this form is used to represent curves in 3D spaces.

 

 

 

The special menu commands available via the attributes menu in the plot windows are the following:

Button

What it does

Variables on axes

opens a window where you can choose the variables or parameters plotted along the axes; there are different options available, depending on the type of curve.
Plotting region opens a window where you can enter the range the different axes.
