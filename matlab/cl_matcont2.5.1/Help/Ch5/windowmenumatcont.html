The window menu in the Main window allows you to open output windows.

Command

What it does

Plot

commands from this submenu (2D-plot or 3D-plot) are used to open plot windows for graphical representation of the computed curves.
Numeric this command is used to open the numeric window.

You can open just one single numeric window but several plot windows.

MATCONT associates opened windows with the current dynamical system and stores their descriptions in the archives. These windows will be opened automatically next time you start MATCONT or select this specific dynamical system.
