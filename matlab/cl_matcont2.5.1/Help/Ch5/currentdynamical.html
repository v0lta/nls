A current dynamical system is one of the dynamical systems in your archives which you currently investigate. All other dynamical systems are inaccesible; you can only deal with one system at a time.

When you create a new dynamical system it becomes the current one. You may also select another already defined system with the systems list dialog box by the select|system command in the Main window.

The name of the current system is displayed in the System information field in the Main window.

MATCONT remembers the current dynamical system between sessions and selects it each time you start MATCONT.
