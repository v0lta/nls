The type menu in the Main window allows you to set set the type of an initial point and the type of the curve to be computed. The menu has two submenus initial point and curve for types of initial point and a curve respectively.

 

MATCONT currently supports the continuation of the following curves:

Orbit
Equilibrium curve
Limit cycle curve

Hopf curve
Limit point cuve

Flip of cycles curve (PeriodDoubling)
Fold of cycles curve (Limit Point of Cycles)
Torus of cycles curve (Neimark-Sacker)

Branch point curve
Branch point of cycles curve

Discrete orbit

Homoclinic to saddle
Homoclinic to saddle-node
