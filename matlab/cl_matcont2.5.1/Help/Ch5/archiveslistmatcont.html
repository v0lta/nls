The archives list dialog box list all subdirectories of the subdirectory for the current system in the Systems directory. These are all subdirectories in which curves can be stored. This list dialog box allows you to manipulate them. You display the archives list dialog box with the select|diagram command in the Main window. The commands listed below operate on the archive directory whose name is highlighted in the list (not only the current curve). To highlight a specific directory just click on its name.

 

Command

What it does
Actions|New creates a new archive directory, for which the name will be asked in a separate input window.
Actions|Rename asks for the new name of the highlighted directory and then renames it. The current curve remains the same. New name may be any text of any length.

Actions|Delete

deletes the highlighted directory, with all the curve-files stored in it.

There are following buttons in the dialog box:

Button

What it does

Select

makes the highlighted archive directory the current one. In particular, when selecting a new curve or initial point, these can only be selected from this new archive directory. This command closes the dialog box.
Cancel This command closes the dialog box leaving the current archive directory unchanged.
