A generator window displays all the parameters of a generator which you may set and change. For each possible combination of initial point and curve there is a specific generator and, correspondingly, a set of parameters which control its performance. After you choose the types of the initial point and the curve to be computed MATCONT loads the appropriate generator and displays the window associated with it.

The number and meaning of the parameters in the window depend on the particular starter but all the windows have the same layout.

The window is divided into sections. A section has the title at the top of it. The title marks the beginning of a section and has no other purpose. There are one or more lines below the title of a section. Each line displays the value of one parameter. The name of a parameter is displayed on the left side of the line while the current value of the parameter is adjusted to its right side.

The way you set or change the value of a parameter depends on the kind of the parameter. There are following kind of parameters

Kind of parameter

Representation and modification of values

Method

specifies the currently selected numerical algorithm used for the computation of the mathematical object (method of integration). This parameter is present only if there are at least two different implementations and it is always in the first line of the window. To change the method, the user needs to click in the line and then choose an item from the drop-down list of available methods.
 
Number values are represented by numbers, integer or float depending on particular parameter. To change the value click it by mouse and then use the keyboard as usual. You have to press the Enter key or move to another input box to indicate that you have completed typing.
 
Exclusive list values are represented by a set of identifiers with each identifier denoting one possible value of a parameter. Only one identifier representing the current value is visible. To change the value click it until the desired value appears.
