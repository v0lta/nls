A starter which produces necessary data to generate the equilibrium curve from a branch point of equilibria has the following parameters that can be modified via the starter window. This starter computes the secondary equilibrium branch. To compute the primary equilibrium branch the user needs to set the type of point to EP, and then simply the  EP_EP starter is used.
