A starter which produces necessary data to generate the torus of cycles curve from an torus bifurcation point (NS) has the following parameters that can be modified via the starter window.

 

Initial point

 

p

a list of parameter names and their values.
period the period of the initial cycle.
Discretization data
ntst the number of test intervals.
ncol the number of collocation points.
Jacobian data
  
Increment Increment to approximate partitial derivatives by finite differences
Switch data
  
Amplitude contains the initial amplitude
Monitor userfunctions
  
function names Monitor toggle {yes/no}
Monitor singularities
  
Singularity types Monitor toggle {yes/no}
Calculate multipliers
  
Multipliers
 Compute toggle {yes/no}
The number of parameters may be greater than the window can accomodate. In such a case use the scroll bar at the right edge of the window to make other parameters visisble.
