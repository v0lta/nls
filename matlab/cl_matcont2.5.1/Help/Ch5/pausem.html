The pause mode dialog box allows you to set the pause mode. You display the dialog box by the Options|Pause command in the Main window.

You can choose one of the following pause modes.

Command

How it works

At each point

suspends computation after each point.
At special points suspends computation after special points only. This is the default mode.
Never does not suspend computation.

There is also the Enable pause buttons check box. This window contains several buttons you may use to suspend and resume computation with the mouse.

During computation of a curve you may control the process with the following buttons.

Key

What it does

Stop

aborts the computation of the current curve.
Pause

suspends computation of the current curve and enters the pause state.

When in a pause (which is indicated by the word 'pause' in the Status field in the Main window) you can press one of the following buttons.

Button

What it does

Resume

resumes computation of the current curve.
Stop aborts the computation of the current curve.
